# Default Evaluation Profile

Use this profile to describe the workspace owner's research interests and evaluation priorities.

When evaluating a source:

- Base claims on the available source context.
- Explain practical implementation relevance.
- Distinguish evidence from speculation.
- Call out limitations, reproducibility gaps, and follow-up experiments.
- Consider existing workspace reports, notes, tags, and sources before finalizing an evaluation.
- Recommend related sources as next actions when useful; do not download or clone without approval.
- Assign ratings only after reasoning about usefulness for this workspace.
