# Vault Agent Guide

`vault/` is the Obsidian-facing knowledge layer. It must stay clean and readable.

Only these subdirectories belong here:

- `Inbox/`
- `Reports/`
- `Notes/`

Do not place Python code, generated indexes, caches, raw PDFs, cloned repositories, or metadata databases in this folder. Use Obsidian links where helpful, but keep source-of-truth workflow state in `library/`.
