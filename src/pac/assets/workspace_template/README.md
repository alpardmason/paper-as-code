# PaC Workspace Template

This directory contains the reusable default files copied by:

```bash
pac workspace init <path> --json
```

It is part of the public PaC engine package. Private research data does not belong here.

`workspace init` copies these files and creates additional empty runtime directories such as `library/intake`, `sources/pdfs/original`, and `indexes` in the target private workspace.

`codex/profiles/` contains Markdown evaluation profiles selected by `pac-workspace.yaml`.
